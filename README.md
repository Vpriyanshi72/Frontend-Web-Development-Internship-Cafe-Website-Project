# Urban Brew Cafe Website

This is a simple and responsive cafe website created for learning and project purposes.

## Features
- Navigation bar with menu links
- Hero section with cafe introduction
- Featured food items
- Image gallery
- Online table booking form
- Customer reviews section
- Contact details and social media icons
- Responsive design using Bootstrap

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome

## Purpose
This project was created to practice web development and design a basic cafe website layout.
